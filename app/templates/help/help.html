    {% trans "Help and FAQ (Frequently Asked Questions)" %}

    You can contact GEN's support by sending an email to {% support_emails %}.

    If you are using a desktop computer or a laptop, a course/module sections list is visible on the left side of the course main page. If the section is unlocked, you can click on it to access its contents. And if you are using a mobile device, the section list is available at the top portion of the page, right under the course/module name. You just need to tap on the Toggle button.

    You can also navigate between sections by using the Previous section and Next section buttons available at the end of the section content.

    Some course/modules might require you to complete another course/module before allowing you to access its contents. The missing requirement should be listed right under course/module name.

    Some sections might require you to complete a requirement before allowing you to access its contents. Please check the section tooltip to check which requirement is missing.

    If your course/module provides a certificate of completion, you should see a Download certificate below the course/module description. If the button is visible but greyed out it means that you have not completed the requirements necessary to mark the course/module as completed and to be able to obtain the certificate.

    To get the best experience with the 3D simulation game we recommend that you use one of the following browsers:

    • Google Chrome
    • Mozilla Firefox
    • Microsoft Edge

    Please note that at this moment the 3D simulation games does not work with mobile devices.
